숫자는 거짓말을 하지 않습니다. 하지만 숫자만 가지고는 상대방을 제대로

#### [생각이 바로 그림이 되는 곳,]{.u2} {#029.xhtml_sigil_toc_id_28 title="생각이 바로 그림이 되는 곳, 제미나이 캔버스"}

[제미나이 캔버스]{.u2}

\

제미나이 대화창에 시각화를 하고 싶은 내용과 데이터를 입력하면 제미나이는
캔버스[Canvas]{.m1}에서 스스로 코딩을 한 후 시각화된 그래프를
보여줍니다. 제미나이 캔버스는 생각한 아이디어가 바로 그림이 되어
돌아오는, 즉시 반응하는 AI 설계실과 같은 공간입니다.

제미나이의 시각화 기능은 크게 두 가지 모델로 나뉩니다. '제미나이 Pro'는
강력한 최신 모델로, 복잡한 데이터 분석과 정교한 상호작용 차트 등 최고
수준의 시각화 작업에 적합합니다. 그리고 '제미나이 Flash[(빠른
모드)]{.m}'는 속도와 비용 효율성에 초점을 맞춰, 간단한 차트를 실시간으로
만들거나 빠른 프로토타입을 제작하는 데 유용합니다.

무료 버전에서도 두 모델을 모두 체험해 볼 수 있지만, 하루 요청 횟수에
따라 사용량 제한이 있어 아이디어를 테스트하거나 학습하는 정도입니다.
하지만 유료 버전은 횟수 제한이 거의 없이 최적화된 성능을 제공합니다.
그래서 방대한 데이터를 다루거나 전문가 수준의 복잡한 인포그래픽을
반복적으로 수정하고 완성해야 하는 실제 업무 및 상업적 목적에는 유료
버전이 필수입니다.

그럼 이제 엑셀 작업 없이 제미나이와 대화하며 꺾은선 그래프를 만들어
보겠습니다. 여기에서는 코딩적 사고를 훈련하는 6단계 프로세스[(① 상상해
보기 → ② 자연어로 말하기 → ③ 결과 미리보기 → ④ 수정 요청하기 → ⑤ 코드
활용하기 → ⑥ 반복하며 구조 익히기)]{.m}를 실습하게 됩니다.

**1단계. 무엇을 만들고 싶은지 그림 그려보기(기획 단계)**

첫 번째 목표는 '연도별 매출 추이'를 한눈에 보는 겁니다. 당신은
'센스일보' 마케팅팀의 데이터 분석 담당자입니다. 지난 5년간의 온라인 광고
매출 변화를 보여주는 그래프를 만들어 상사에게 보고하려고 합니다.

우선 그래프의 완성된 모습을 머릿속에 그려보세요. 시간이 흐를수록 점점
상승하는 곡선, 특정 시점에서 급격한 변화가 일어나는 지점 표시, 이 모든
것이 '숫자가 말하게 만드는' 첫걸음입니다.

\

**2단계. 제미나이에게 자연어로 말하기(요청 단계)**

이제 제미나이에 접속해 프로젝트 디렉터로서 첫 번째 지시를 내려
보겠습니다. 코딩은 몰라도 괜찮습니다. 우리가 원하는 것을 명확하고
구체적인 언어로 전달하는 것이 핵심입니다.

\

::: tb1
[프롬프트 예시]{.cor1}

\

2021년 100억, 2022년 125억, 2023년 130억, 2024년 145억, 2025년 160억의
온라인 광고 매출 데이터를 가지고, 연도별 변화를 보여주는 꺾은선 그래프를
만들어줘. 그래프 제목은 '센스일보 5년간 온라인 광고 매출 추이'로 해줘.
:::

\

제미나이 대화창에 프롬프트를 입력하고, 대화창 하단의 도구를 클릭해
'Canvas' 기능을 활성화시켜 주세요. 그리고 모델은 '빠른 모드'로 선택하고
'실행' 버튼을 누릅니다.

**3단계. 생성된 코드를 실시간으로 확인하기(오류 찾기)**

자, 이제 제미나이가 코드를 생성하는 모습을 실시간으로 볼 수 있습니다.
코드를 생성하는 화면 우측 상단의 '미리보기' 버튼을 누르면 완성된
결과물을 확인할 수 있습니다.

그런데 어색한 부분이 보입니다. 그래프의 왼쪽과 하단의 수치가 잘려서
표시되었습니다. 괜찮습니다. 이것이 바로 '코딩적 사고' 훈련의 시작입니다.
우리는 이 오류를 보고 당황하지 말고, AI에게 다시 정확하게 지시하면
됩니다. 마치 신입사원이 말을 알아듣지 못할 때, 차근차근 다시 설명해 주는
것과 같죠.

**4단계. 제미나이에게 1차 수정 요청하기(문제해결)**

그럼 이제 제미나이가 스스로 오류를 수정하도록 유도하거나, 우리가 직접
정확하게 데이터를 수정해 줘야 합니다. 그래프에서 잘못 표시된 내용의
수정사항을 왼쪽 프롬프트 창에 요청하면 제미나이는 즉시 코드를 수정하여
깔끔하게 정리된 그래프를 보여줍니다.

\

::: tb1
[프롬프트 예시]{.cor1}

넉넉하게 조정해 모든 수치가 또렷하게 보이도록 수정해 줘. 각 데이터

가독성만 더 좋아지게 정리해 줘.

**5단계. 제미나이에게 2차 수정 요청하기(스토리 추가)**

이제 그래프는 어느 정도 완성되었습니다. 그런데 '왜' 매출이 변했는지 아직
스토리가 없습니다. 우리는 이 그래프에 매출 급증에 대한 '스토리'를 입혀야
합니다. 제미나이에게 다음과 같이 2차 수정을 요청하면 제미나이는 코드를
다시 수정하여 2022년과 2024년 지점에 작은 텍스트로
주석[Annotation]{.m1}을 표시합니다.

\

::: tb1
[프롬프트 예시]{.cor1}

그리고 2024년 매출 상승에는 'AI 솔루션 도입'이라고 주석을 달아줘.

**6단계. 제미나이에게 3차 수정 요청하기(시각적 강조)**

그런데 주석이 달리긴 했지만 글씨가 작고 눈에 잘 띄지 않아 상사가 한눈에
알아보지 못할 것 같습니다. 이 중요한 포인트를 어떻게 하면 '눈에 확 띄게'
만들 수 있을까요?

이것이 바로 '코딩적 사고'의 핵심입니다. 우리는 제미나이에게 단순히
'예쁘게 해줘'가 아니라 '어떻게' 예쁘게 할지 구체적으로 지시해야 합니다.
여기에서는 ACTS 프레임워크를 사용해 프롬프트 아이디어를 얻어보도록
하겠습니다.

\

::: tb1
[프롬프트 예시]{.cor1}

\

**A(Actor -- 역할)** : 너는 비즈니스 스토리텔링 디자이너이자 데이터
시각화 전문가야. 이 차트의 목적은 '2022년과 2024년의 전략적 투자가
이후의 폭발적인 매출 성장을 이끌었다'는 핵심메시지를 경영진에게 단번에
설득하는 거야.

**C(Context -- 맥락)** : 이 차트는 센스일보의 5년간 온라인 광고 매출
추이를 보여주는 꺾은선 그래프야. 단순히 수치만 보여주는 게 아니라,
전략적 성공 스토리가 한눈에 보이도록 2022년과 2024년의 매출 급증 이유를
시각적으로 확실하게 강조하고 싶어.

**T(Task -- 작업)** :

\- 2022년 125억과 2024년 145억 데이터 포인트를 다른 점들보다 훨씬 크고
강렬한 주황색[(#FF4500)]{.m}으로 표시해서 눈에 확 띄게 해줘.

\- 주석은 밝은 배경색이 있는 박스 안에 넣고, 강조된 점까지 주황색의 굵고
선명한 화살표를 그려서 연결해 줘.

\- 2022년 이후와 2024년 이후의 라인을 기존보다 훨씬 굵고 밝은
녹색[(#00BFA5)]{.m}으로 칠해서 성장 모멘텀이 계속되고 있다는 걸 보여줘.

**S(Style -- 스타일)** : 전체적으로 신뢰감을 주는 파란색 톤을 유지하되,
핵심내용에는 주황색과 녹색을 써서 반전 있는 대비 효과를 만들어줘.
불필요한 그리드선은 싹 지워서 깔끔하고 명확하게 처리해 줘.

**(1) 웹 링크(URL)로 즉시 공유하기**

가장 빠르고 가벼운 공유 방식입니다. 복잡하게 파일을 주고받을 필요 없이,
인터넷 주소만으로 콘텐츠를 전달합니다.

인포그래픽 우측 상단의 '공유'를 누르면 '링크 복사' 창이 활성화됩니다.
여기서 생성된 링크 주소를 복사해 상대방에게 전달하면 PC 또는 모바일 웹
브라우저를 통해 인포그래픽을 즉시 확인할 수 있습니다.

\

::: w100
![](AI로 차트 그리기/Images/176.jpg){.w100}
:::

\

**(2) PDF 파일로 저장하여 문서화하기**

문서 보고용이나 출력용 자료가 필요할 때에는 웹 브라우저의 기본 기능을
사용하면 됩니다. 링크를 열고, 브라우저의 인쇄 기능[(Ctrl + P 또는 Cmd +
P)]{.m}에서 'PDF로 저장'을 선택해 파일로 저장합니다.

\

**(3) 이미지 파일(PNG)로 저장하기**

별도의 저장 버튼이 없을 경우, 가장 빠르고 확실하게 이미지를 확보하려면
컴퓨터나 스마트폰에서 제공하는 화면 캡처[(스크린샷)]{.m} 기능을 사용해
인포그래픽 영역만 잘라내어 이미지 파일로 저장합니다. 이미지로 저장하면
PPT 슬라이드에 삽입하거나 채팅방에서 간편하게 공유할 수 있습니다.
