## 문제 0477

$f(x) = x^{50} + \dfrac{1}{x^{50}}$일 때, $f\!\left(\dfrac{1-i}{1+i}\right)$의 값을 구하시오. (단, $i = \sqrt{-1}$)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 복소수 분모 유리화 | $\dfrac{a+bi}{c+di} \cdot \dfrac{c-di}{c-di}$ (켤레복소수 곱) |
| $i$의 거듭제곱 주기 | $i^4 = 1$ (주기 4), $i^{4k+r} = i^r$ |
| 음수의 짝수 제곱 | $(-1)^{2k} = 1$ |

### 풀이방법

1. **$\dfrac{1-i}{1+i}$ 간단히 하기**:

$$\frac{1-i}{1+i} = \frac{(1-i)(1-i)}{(1+i)(1-i)} = \frac{(1-i)^2}{1^2+1^2} = \frac{1-2i+i^2}{2} = \frac{1-2i-1}{2} = \frac{-2i}{2} = -i$$

2. **$x = -i$로 놓고 $x^{50}$ 계산**:

$$x^{50} = (-i)^{50} = (-1)^{50} \cdot i^{50} = 1 \cdot i^{50}$$

$$i^{50} = i^{4 \times 12 + 2} = (i^4)^{12} \cdot i^2 = 1 \cdot (-1) = -1$$

$$\therefore\; x^{50} = -1$$

3. **$\dfrac{1}{x^{50}}$ 계산**:

$$\frac{1}{x^{50}} = \frac{1}{-1} = -1$$

4. **결과**:

$$f\!\left(\frac{1-i}{1+i}\right) = (-1) + (-1) = \boxed{-2}$$

> **오답 포인트**: $(-i)^{50}$을 계산할 때 $(-1)^{50}$과 $i^{50}$을 분리하지 않고 $(-i)^{50} = -i^{50}$으로 처리하는 실수. $(-i)^{50} = (-1)^{50} \cdot i^{50} = i^{50}$임에 주의.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $\dfrac{a+bi}{c+di}$ 형태의 복소수를 간단히 하는 방법은?

**A.** 분모의 켤레복소수 $c-di$를 분자·분모에 곱한다. 분모는 $c^2+d^2$ (실수)가 된다.

### 카드 2
**Q.** $i^n$의 값을 빠르게 구하는 방법은?

**A.** $n$을 4로 나눈 나머지 $r$을 구한다. $i^0=1,\ i^1=i,\ i^2=-1,\ i^3=-i$ 중 $i^r$을 읽으면 된다.

### 카드 3
**Q.** $(-z)^n$에서 $n$이 짝수일 때 $(-1)^n$은?

**A.** $(-1)^n = 1$ (짝수 거듭제곱). 따라서 $(-z)^n = z^n$.

### 카드 4
**Q.** $f(x) = x^n + \dfrac{1}{x^n}$에서 $x^n$의 값을 구했을 때 함숫값을 계산하는 방법은?

**A.** $x^n = c$이면 $\dfrac{1}{x^n} = \dfrac{1}{c}$이므로 $f(x) = c + \dfrac{1}{c}$. 특히 $c = -1$이면 $f(x) = -1 + (-1) = -2$.

### 마지막 카드
**Q.** 복소수 거듭제곱 계산에서 자주 하는 실수(오답 패턴)는?

**A.** ① $(-i)^n$을 $-(i^n)$으로 잘못 처리. 반드시 $(-1)^n \cdot i^n$으로 분리. ② $i^n$의 주기를 4가 아닌 2로 착각. ③ 분모 유리화 후 $(1-i)^2 = 1-2i-1$에서 $i^2=-1$을 빠뜨리는 계산 실수.
