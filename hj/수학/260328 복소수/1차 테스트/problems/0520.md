# 복소수 오답 문제 정리

> 원본 이미지: `SCR-20260328-nsdl.png`
> ⚠️ 이미지 해상도가 낮아 조건의 우변($8+12i$)이 불명확할 수 있음

---

## 문제 0520

$a_1, a_2, a_3, \ldots, a_{30}$은 각각 $-1,\, 1,\, i,\, -i$ 중 하나의 값을 가질 때,

$$a_1^2 + a_2^2 + a_3^2 + \cdots + a_{30}^2 = 8 + 12i$$

를 만족시킬 때, $a_1 + a_2 + \cdots + a_{30}$의 실수부분과 허수부분의 합을 구하시오.

(단, $i = \sqrt{-1}$)

> ※ 추정: $a_k^2 \in \{1, -1\}$ (실수)이므로 합이 $8+12i$가 되려면 모순이 발생함. 이미지의 조건 우변이 실제로는 다를 가능성 있음. 아래에서는 **가능한 두 가지 해석**으로 풀이를 제공함.

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| $a_k$ 값 | $a_k^2$ |
|-----------|----------|
| $1$ | $1$ |
| $-1$ | $1$ |
| $i$ | $-1$ |
| $-i$ | $-1$ |

### 풀이방법

각 값의 개수를 다음과 같이 정의:
- $p$: $a_k = 1$인 개수
- $q$: $a_k = -1$인 개수
- $r$: $a_k = i$인 개수
- $s$: $a_k = -i$인 개수

**조건 ①** (총 개수):
$$p + q + r + s = 30$$

**조건 ②** (제곱합):
$$\sum a_k^2 = (p+q)\cdot 1 + (r+s)\cdot(-1) = (p+q) - (r+s)$$

$a_k^2$의 합은 항상 **순수 실수**이므로, 우변의 허수부가 $0$이어야 한다.

**해석 A**: 우변이 $8$ (실수)인 경우:
$$(p+q) - (r+s) = 8 \quad \text{and} \quad p+q+r+s = 30$$
$$\Rightarrow p+q = 19, \quad r+s = 11$$

$$\sum a_k = (p - q) + (r - s)i$$

실수부 $p-q$와 허수부 $r-s$는 추가 조건 없이 유일하게 결정되지 않음. 추가 조건이 더 있을 것으로 추정.

**해석 B**: 문제 조건이 $\sum a_k^2 = 8$이고, 추가로 $a_1 + \cdots + a_{30}$의 실수부 $+$ 허수부를 구하는 경우:

$p+q = 19$, $r+s = 11$ 확정. $\sum a_k = (p-q) + (r-s)i$.
- 실수부: $p-q$ — 범위 $-19 \leq p-q \leq 19$ (홀수)
- 허수부: $r-s$ — 범위 $-11 \leq r-s \leq 11$ (홀수)

최솟값·최댓값 등 추가 조건이 있을 때 결정 가능.

> ※ 원본 문제의 정확한 조건 확인 후 풀이 완성 필요.

> **오답 포인트**: $a_k$의 합과 $a_k^2$의 합을 혼동하는 것. 제곱합 조건으로 각 값의 개수를 구한 뒤, 별도로 일차합을 계산해야 한다. 또한 $a_k^2$의 합이 허수가 될 수 없다는 사실을 간과하면 조건이 모순임을 파악하지 못한다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $a_k \in \{1, -1, i, -i\}$일 때 $a_k^2$가 가질 수 있는 값은?

**A.** $1^2 = 1$, $(-1)^2 = 1$, $i^2 = -1$, $(-i)^2 = -1$. 즉 $a_k^2 \in \{1, -1\}$로 항상 실수이다.

---

### 카드 2
**Q.** 복소수 값을 갖는 수열에서 각 값의 개수를 변수로 놓고 연립하는 방법은?

**A.** 가능한 값의 종류마다 개수 변수($p, q, r, s, \ldots$)를 설정한 뒤,
① 총 개수 방정식 $\sum = n$,
② 합(일차) 방정식 — 실수부·허수부 따로,
③ 제곱합 방정식
을 연립하여 각 개수를 결정한다.

---

### 카드 3
**Q.** $a_k \in \{1, -1, i, -i\}$일 때 $\sum a_k$의 실수부와 허수부를 각 개수로 나타내면?

**A.** 각 값의 개수를 $p, q, r, s$라 하면:
$$\sum a_k = p\cdot 1 + q\cdot(-1) + r\cdot i + s\cdot(-i) = (p-q) + (r-s)i$$
- 실수부: $p - q$
- 허수부: $r - s$

---

### 카드 4
**Q.** 복소수 합의 조건식(등식/부등식)이 주어질 때, 실수·허수 조건을 분리하여 연립하는 핵심 원칙은?

**A.** 복소수 등식 $A = B$는 $\text{Re}(A) = \text{Re}(B)$이고 $\text{Im}(A) = \text{Im}(B)$. 두 방정식을 독립적으로 세워 연립한다. 한쪽 조건만 사용하면 해가 과도하게 결정되거나 부족해진다.

---

### 카드 5
**Q.** 이 유형 문제에서 자주 하는 실수(오답 패턴)는?

**A.**
- $\sum a_k^2$와 $\sum a_k$를 혼동하여 같은 값으로 처리
- $a_k^2$의 합이 항상 실수임을 몰라, 허수 조건이 있을 때 모순 탐지 실패
- 개수 변수 연립 시 실수부·허수부 방정식을 따로 세우지 않고 하나의 방정식만 사용
- 실수부와 허수부의 "합"을 구하라는 문제에서 둘 중 하나만 구하는 마무리 실수
