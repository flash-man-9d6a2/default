## 문제 0523

2015년 06월 고1 하학평가 27번

등식

$$(i+i^2)+(i^3+i^4)+(i^5+i^6)+\cdots+(i^{19}+i^{20}) = a+bi$$

를 만족시키는 실수 $a$, $b$에 대하여 $4(a+b)^2$의 값을 구하시오.

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| $i$의 4주기 | $i^1=i,\ i^2=-1,\ i^3=-i,\ i^4=1$ |
| 연속 두 항 묶음 | $i^{2k-1}+i^{2k}$는 $k$ 홀수: $-1+i$, $k$ 짝수: $1-i$ |
| 인접 두 묶음 합 | $(-1+i)+(1-i) = 0$ |

### 풀이방법

1. **각 괄호 값 계산**:

| 묶음 | 계산 | 결과 |
|------|------|------|
| $i+i^2$ | $i+(-1)$ | $-1+i$ |
| $i^3+i^4$ | $(-i)+1$ | $1-i$ |
| $i^5+i^6$ | $i+(-1)$ | $-1+i$ |
| $i^7+i^8$ | $(-i)+1$ | $1-i$ |
| $\vdots$ | $\vdots$ | $\vdots$ |
| $i^{19}+i^{20}$ | $(-i)+1$ | $1-i$ |

2. **패턴 확인**: $(-1+i)$와 $(1-i)$가 교대 반복. 인접 두 묶음의 합 $= (-1+i)+(1-i) = 0$.

3. **총 묶음 수**: 지수 $1$~$20$을 2개씩 묶으면 **10묶음** → 5쌍.

$$\text{전체 합} = 5 \times [(-1+i)+(1-i)] = 5 \times 0 = 0$$

4. **결론**:

$$a+bi = 0 \implies a = 0,\quad b = 0$$

$$4(a+b)^2 = 4(0+0)^2 = \boxed{0}$$

> **오답 포인트**: 묶음 수를 잘못 세어 홀수 개의 묶음이 남는다고 착각하거나, $(-1+i)+(1-i) = 0$임을 놓치고 각 항을 따로 더하다 계산 실수를 하는 경우.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $i$의 거듭제곱에서 $i^n$의 값을 빠르게 구하는 방법은?
**A.** $i$는 4주기: $i^1=i,\ i^2=-1,\ i^3=-i,\ i^4=1$. $n$을 4로 나눈 나머지 $r$에 대해 $i^n = i^r$ ($r=0$이면 $1$).

### 카드 2
**Q.** $i$의 거듭제곱의 합 $i^1+i^2+\cdots+i^n$을 효율적으로 계산하는 방법은?
**A.** 4개씩 묶으면 $i+(-1)+(-i)+1 = 0$이므로, 4의 배수 단위로 묶어 0으로 소거한 뒤 남은 항만 계산한다. 또는 연속 두 항끼리 묶는 패턴($i^{2k-1}+i^{2k}$)을 활용한다.

### 카드 3
**Q.** 복소수의 합이 $a+bi$ 형태로 주어졌을 때 $a$, $b$를 구하는 방법은?
**A.** 합산 결과의 실수부 $= a$, 허수부의 계수 $= b$로 분리한다. 합산 중간에 허수 성분과 실수 성분을 각각 따로 모아도 된다.

### 카드 4
**Q.** $i^{2k-1}+i^{2k}$ ($k = 1, 2, 3, \ldots$)의 값이 어떻게 교대하는가?
**A.** $k$ 홀수: $i^{2k-1}+i^{2k} = i+(-1) = -1+i$. $k$ 짝수: $i^{2k-1}+i^{2k} = (-i)+1 = 1-i$. 인접한 두 값의 합은 $0$.

### 카드 5
**Q.** $i$의 거듭제곱 합산 문제에서 자주 하는 실수(오답 패턴)는?
**A.** ① 4주기를 2주기로 착각하는 것. ② 연속 두 항의 합이 0이 되는 쌍 패턴을 활용하지 못하고 모든 항을 개별 계산하다 실수. ③ 마지막 지수가 4의 배수인지 확인하지 않아 남은 항을 빠뜨리거나 잘못 처리하는 것.
