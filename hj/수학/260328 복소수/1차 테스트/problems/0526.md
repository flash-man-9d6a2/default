## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 켤레복소수의 합 | $z + \bar{z} = 2\,\text{Re}(z)$ (실수) |
| 켤레복소수의 곱 | $z \cdot \bar{z} = \lvert z \rvert^2$ (실수) |
| 켤레복소수 분배 | $\overline{z_1 + z_2} = \bar{z}_1 + \bar{z}_2$ |
| 켤레복소수 곱 분배 | $\overline{z_1 z_2} = \bar{z}_1 \bar{z}_2$ |
| 실수의 켤레 | $z$가 실수 $\implies \bar{z} = z$ |

### 풀이방법

> ※ 이미지 해상도 문제로 정확한 풀이를 제공할 수 없다. 아래는 이 유형의 일반적인 풀이 전략이다.

복소수 $z_1, z_2, z_3, z_4$에 대한 조건이 주어지고 그 곱이나 합을 구하는 유형의 일반 접근:

1. 켤레 관계($z_3 = \bar{z}_1$ 등)를 파악하여 식을 단순화한다.
2. 켤레 쌍의 합 $z_k + \bar{z}_k = 2\,\text{Re}(z_k)$ (실수), 곱 $z_k \bar{z}_k = \lvert z_k \rvert^2$ (실수)임을 활용.
3. 주어진 등식(예: 합의 곱 $= $ 상수)에서 미지수를 연립하여 푼다.
4. 실수부와 허수부를 분리하여 방정식을 세운다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** 켤레복소수 쌍 $z$와 $\bar{z}$의 합과 곱은 어떤 성질을 가지는가?
**A.** 합: $z + \bar{z} = 2\,\text{Re}(z)$ → 항상 실수. 곱: $z \cdot \bar{z} = \lvert z \rvert^2$ → 항상 비음의 실수. 켤레 쌍이 관여하면 결과가 실수가 되어 계산이 단순해진다.

### 카드 2
**Q.** 여러 복소수의 합이나 곱에 켤레 연산을 적용하는 방법은?
**A.** $\overline{z_1 + z_2 + \cdots + z_n} = \bar{z}_1 + \bar{z}_2 + \cdots + \bar{z}_n$ (선형성). $\overline{z_1 z_2 \cdots z_n} = \bar{z}_1 \bar{z}_2 \cdots \bar{z}_n$ (곱의 분배). 이를 이용해 복잡한 식을 분해한다.

### 카드 3
**Q.** 복소수들의 실수부·허수부 조건에서 연립방정식을 세우는 방법은?
**A.** $z = a+bi$로 놓고 주어진 조건 식을 전개한 뒤, 좌변과 우변의 실수부는 실수부끼리, 허수부는 허수부끼리 같다고 놓아 연립방정식을 세운다.

### 카드 4
**Q.** $(z_1 + z_2)(z_3 + z_4)$를 계산할 때 켤레 관계를 어떻게 활용하는가?
**A.** $z_3 = \bar{z}_1$, $z_4 = \bar{z}_2$이면 $z_3 + z_4 = \overline{z_1 + z_2}$이므로 $(z_1+z_2)(z_3+z_4) = (z_1+z_2)\overline{(z_1+z_2)} = \lvert z_1+z_2 \rvert^2$. 결과가 실수이다.

### 카드 5
**Q.** 복소수 관계식 문제에서 자주 하는 실수(오답 패턴)는?
**A.** ① 켤레 관계를 활용하지 않고 각 복소수를 독립적으로 계산하는 것. ② 켤레 쌍의 합·곱이 실수라는 사실을 놓치는 것. ③ $(a+bi)(a-bi) = a^2+b^2$임을 $a^2-b^2$으로 잘못 계산하는 부호 실수.
