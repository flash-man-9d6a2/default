## 문제 0412

$x + 5y = 1$을 만족하는 두 실수 $x$, $y$에 대하여 등식

$$x + yi = \frac{-1}{1 + ai}$$

가 성립할 때, 양수 $a$의 값은? (단, $i = \sqrt{-1}$)

> ※ 추정: 좌변이 $x + yi$, 우변 분모가 $1 + ai$로 추정됨.

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 복소수 분모 유리화 | $\dfrac{c}{a+bi} = \dfrac{c(a-bi)}{a^2+b^2}$ |
| 켤레복소수 곱 | $(a+bi)(a-bi) = a^2 + b^2$ (실수) |
| 복소수 등치 조건 | $x + yi = p + qi \iff x = p,\; y = q$ ($x,y,p,q$ 실수) |

### 풀이방법

1. **우변 유리화** — 분모의 켤레복소수 $1 - ai$를 곱함:

$$\frac{-1}{1+ai} = \frac{-1 \cdot (1-ai)}{(1+ai)(1-ai)} = \frac{-1+ai}{1+a^2}$$

$$= \frac{-1}{1+a^2} + \frac{a}{1+a^2}i$$

2. **복소수 등치 조건 적용** ($x, y$는 실수):

$$x = \frac{-1}{1+a^2}, \qquad y = \frac{a}{1+a^2}$$

3. **$x + 5y = 1$ 대입**:

$$\frac{-1}{1+a^2} + \frac{5a}{1+a^2} = 1$$

$$\frac{-1 + 5a}{1+a^2} = 1$$

$$-1 + 5a = 1 + a^2$$

$$a^2 - 5a + 2 = 0$$

4. **근의 공식**:

$$a = \frac{5 \pm \sqrt{25 - 8}}{2} = \frac{5 \pm \sqrt{17}}{2}$$

5. **양수 조건**: $\sqrt{17} \approx 4.12$이므로

$$a = \frac{5 + \sqrt{17}}{2} \approx 4.56 \quad \text{또는} \quad a = \frac{5 - \sqrt{17}}{2} \approx 0.44$$

두 근 모두 양수이므로, 문제에서 추가 조건이 있는 경우 원본 확인이 필요하다.

> **오답 포인트**: 유리화 시 분모 $(1+ai)(1-ai) = 1 + a^2$임에 주의. $i^2 = -1$이므로 $-a^2 i^2 = a^2$이 되어 결과가 $a^2 - (-a^2)$이 아닌 $1 + a^2$이다. 또한 유리화 후 반드시 실수부/허수부를 $x$, $y$와 등치시켜야 하며, 바로 $x + 5y$ 조건을 분수 형태에 대입하려는 실수를 주의한다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $\dfrac{c}{a+bi}$ 형태를 $x + yi$ 꼴로 변환하는 절차는?

**A.** 분모의 켤레복소수 $a - bi$를 분자·분모에 곱한다:
$$\frac{c(a-bi)}{(a+bi)(a-bi)} = \frac{c(a-bi)}{a^2+b^2} = \frac{ca}{a^2+b^2} - \frac{cb}{a^2+b^2}i$$

---

### 카드 2
**Q.** $(a+bi)(a-bi)$를 전개하면? 왜 허수부가 사라지는가?

**A.** $a^2 - (bi)^2 = a^2 - b^2 i^2 = a^2 + b^2$ (실수). $i^2 = -1$이므로 $-b^2 i^2 = +b^2$이 되어 허수부가 소거된다.

---

### 카드 3
**Q.** 복소수 등식 $x + yi = p + qi$ ($x, y, p, q \in \mathbb{R}$)에서 $x$, $y$를 구하는 원리는?

**A.** 실수부끼리, 허수부끼리 같아야 한다: $x = p$, $y = q$. 이 원리는 복소수의 相等 조건(실수부 동치, 허수부 동치)에서 나온다.

---

### 카드 4
**Q.** 복소수를 포함한 방정식을 실수 조건으로 변환할 때의 일반적 전략은?

**A.** ① 복소수 분수를 유리화하여 $p + qi$ 형태로 정리 → ② 실수부 $= x$, 허수부 $= y$ 등치 → ③ 추가 실수 조건(예: $x + 5y = 1$) 대입 → ④ 방정식 풀기

---

### 카드 5
**Q.** 이 유형에서 자주 하는 실수(오답 패턴)는?

**A.**
- $(a+bi)(a-bi) = a^2 - b^2$으로 계산하는 오류 ($i^2 = -1$ 적용 누락).
- 유리화 후 실수부/허수부를 $x, y$에 등치시키지 않고 바로 조건식에 분수를 대입하여 통분이 복잡해지는 실수.
- 근의 공식 결과에서 양수 조건을 확인하지 않고 두 근 중 하나만 답으로 쓰는 실수.
