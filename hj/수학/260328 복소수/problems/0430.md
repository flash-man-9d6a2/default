## 문제 0430

2022년 06월 고1 학력평가 9번

$x = 2+i$, $y = 2-i$일 때, $x^4 + x^2y^2 + y^4$의 값은? (단, $i = \sqrt{-1}$)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 대칭식 변형 | $x^4 + x^2y^2 + y^4 = (x^2+y^2)^2 - x^2y^2$ |
| 켤레복소수 곱 | $(a+bi)(a-bi) = a^2 + b^2$ (실수) |
| 켤레복소수 제곱합 | $(a+bi)^2 + (a-bi)^2 = 2(a^2 - b^2)$ (실수) |

### 풀이방법

**전략:** $x = 2+i$, $y = 2-i$는 켤레복소수 쌍이므로, $xy$와 $x^2+y^2$을 먼저 계산하면 효율적이다.

1. **$xy$ 계산** (켤레복소수의 곱):

$$xy = (2+i)(2-i) = 4 - i^2 = 4 + 1 = 5$$

2. **$x^2y^2$ 계산**:

$$x^2y^2 = (xy)^2 = 5^2 = 25$$

3. **$x^2$, $y^2$ 계산**:

$$x^2 = (2+i)^2 = 4 + 4i + i^2 = 3 + 4i$$
$$y^2 = (2-i)^2 = 4 - 4i + i^2 = 3 - 4i$$
$$x^2 + y^2 = 6$$

4. **대칭식 변형 적용**:

$$x^4 + x^2y^2 + y^4 = (x^2+y^2)^2 - x^2y^2 = 6^2 - 25 = 36 - 25 = \boxed{11}$$

> **오답 포인트**: $x^4$, $y^4$를 직접 계산하려 하면 $(3+4i)^2 = 9 + 24i + 16i^2 = -7 + 24i$ 등 복잡해진다. $(x^2+y^2)^2 - x^2y^2$로 변형하는 것이 핵심이며, 이때 $(x^2+y^2)^2 = x^4 + 2x^2y^2 + y^4$이므로 반드시 $x^2y^2$을 빼야 $x^4 + x^2y^2 + y^4$가 된다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $x^4 + x^2y^2 + y^4$를 $(x^2+y^2)$과 $xy$로 표현하는 방법은?

**A.** $(x^2+y^2)^2 = x^4 + 2x^2y^2 + y^4$이므로,
$$x^4 + x^2y^2 + y^4 = (x^2+y^2)^2 - x^2y^2$$

---

### 카드 2
**Q.** $x = a+bi$, $y = a-bi$ (켤레복소수 쌍)일 때, $xy$와 $x^2+y^2$의 값은?

**A.**
- $xy = (a+bi)(a-bi) = a^2 + b^2$ (실수)
- $x^2 + y^2 = (a+bi)^2 + (a-bi)^2 = 2(a^2 - b^2)$ (실수)

켤레복소수 쌍의 기본 대칭량은 항상 실수가 된다.

---

### 카드 3
**Q.** 복소수를 포함한 대칭식 $f(x, y)$를 효율적으로 계산하는 전략은?

**A.** 대칭식의 기본량 $x+y$, $xy$를 먼저 구한 뒤, 항등식을 이용해 $f(x,y)$를 기본량으로 표현한다. 직접 $x^n$을 계산하는 것보다 훨씬 간결하다.

---

### 카드 4
**Q.** $(x^2+y^2)^2$과 $x^4 + x^2y^2 + y^4$의 차이는?

**A.** $(x^2+y^2)^2 = x^4 + 2x^2y^2 + y^4$이므로, $x^4 + x^2y^2 + y^4$보다 $x^2y^2$만큼 크다. 즉,
$$x^4 + x^2y^2 + y^4 = (x^2+y^2)^2 - x^2y^2$$

---

### 카드 5
**Q.** 이 유형에서 자주 하는 실수(오답 패턴)는?

**A.**
- $x^4 + x^2y^2 + y^4 = (x^2+y^2)^2$으로 착각하여 $x^2y^2$을 빼지 않는 실수.
- $xy$를 구하지 않고 $x^2$, $y^2$을 각각 계산한 뒤 $x^4$, $y^4$까지 직접 전개하여 복잡해지는 실수.
- $(a+bi)^2$에서 $i^2 = -1$ 적용을 놓쳐 부호를 틀리는 실수.
