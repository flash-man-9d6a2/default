## 문제 0436

두 복소수 $\alpha$, $\beta$에 대하여 $\alpha\bar{\alpha} = \beta\bar{\beta} = 10$, $\alpha + \beta = 2i$일 때, $\alpha\beta$의 값은?
(단, $i = \sqrt{-1}$이고 $\bar{\alpha}$, $\bar{\beta}$는 각각 $\alpha$, $\beta$의 켤레복소수이다.)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 켤레복소수와 절댓값 제곱 | $z\bar{z} = \|z\|^2$ |
| 켤레복소수로 역수 표현 | $z\bar{z} = k \implies \bar{z} = \dfrac{k}{z}$ |
| 켤레의 선형성 | $\overline{\alpha + \beta} = \bar{\alpha} + \bar{\beta}$ |
| 켤레 연산 | $\overline{ai} = -ai$ (순허수의 켤레는 부호 반전) |

### 풀이방법

**전략:** $\bar{\alpha}$, $\bar{\beta}$를 $\alpha$, $\beta$의 분수로 표현한 뒤, $\alpha + \beta$의 켤레 등식을 이용한다.

1. **$\bar{\alpha}$, $\bar{\beta}$를 역수 형태로 표현**:

$$\alpha\bar{\alpha} = 10 \implies \bar{\alpha} = \frac{10}{\alpha}$$
$$\beta\bar{\beta} = 10 \implies \bar{\beta} = \frac{10}{\beta}$$

2. **$\alpha + \beta = 2i$의 양변에 켤레 연산 적용**:

$$\overline{\alpha + \beta} = \overline{2i}$$
$$\bar{\alpha} + \bar{\beta} = -2i$$

3. **1번 결과를 대입**:

$$\frac{10}{\alpha} + \frac{10}{\beta} = -2i$$

4. **좌변 통분**:

$$\frac{10(\alpha + \beta)}{\alpha\beta} = -2i$$

5. **$\alpha + \beta = 2i$ 대입**:

$$\frac{10 \cdot 2i}{\alpha\beta} = -2i$$

$$\frac{20i}{\alpha\beta} = -2i$$

6. **$\alpha\beta$ 계산**:

$$\alpha\beta = \frac{20i}{-2i} = \boxed{-10}$$

> **오답 포인트**: $\alpha + \beta = 2i$의 양변에 켤레를 취할 때, $\overline{2i} = -2i$임을 놓치고 $\bar{\alpha} + \bar{\beta} = 2i$로 두는 실수가 가장 흔하다. 순허수 $ai$의 켤레는 $-ai$로 부호가 반전된다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $z\bar{z} = k$ ($k$는 실수)일 때, $\bar{z}$를 $z$로 표현하면?

**A.** $\bar{z} = \dfrac{k}{z}$. 이를 이용하면 켤레복소수를 역수 형태로 바꾸어 대수적으로 처리할 수 있다.

---

### 카드 2
**Q.** 켤레 연산의 선형성 공식과 활용법은?

**A.** $\overline{\alpha + \beta} = \bar{\alpha} + \bar{\beta}$, $\overline{\alpha\beta} = \bar{\alpha}\bar{\beta}$. 주어진 등식의 양변에 켤레를 취해 새로운 방정식을 얻는 데 활용한다.

---

### 카드 3
**Q.** 다음을 계산하라: $\overline{a + bi}$, $\overline{bi}$, $\overline{a}$ ($a, b \in \mathbb{R}$)

**A.**
- $\overline{a + bi} = a - bi$
- $\overline{bi} = -bi$ (순허수의 켤레는 부호 반전)
- $\overline{a} = a$ (실수의 켤레는 자기 자신)

---

### 카드 4
**Q.** $|\alpha|^2 = |\beta|^2 = k$이고 $\alpha + \beta$가 주어질 때, $\alpha\beta$를 구하는 전형적인 전략은?

**A.** ① $\bar{\alpha} = k/\alpha$, $\bar{\beta} = k/\beta$로 치환 → ② $\alpha+\beta$에 켤레 연산 적용하여 $\bar{\alpha}+\bar{\beta}$ 등식 획득 → ③ 대입 후 통분하여 $\dfrac{k(\alpha+\beta)}{\alpha\beta} = \overline{(\alpha+\beta)}$ 형태로 만들기 → ④ $\alpha\beta$ 계산

---

### 카드 5
**Q.** 이 유형에서 자주 하는 실수(오답 패턴)는?

**A.**
- 순허수 $2i$의 켤레를 $2i$로 두는 실수 ($\overline{2i} = -2i$임에 주의).
- $\bar{\alpha} = 10/\alpha$를 구한 뒤 통분 단계에서 $\alpha + \beta$를 대입하는 것을 빠뜨리는 실수.
- $z\bar{z} = |z|^2$임을 모르고 $\bar{\alpha}$를 $\alpha$의 공액으로 성분 분해하여 연립방정식을 세우는 비효율적 접근.
