## 문제 0465

0이 아닌 복소수 $z = (x^2 - 26) + (5x^2 - 24x - 5)i$에 대하여

$z = \bar{z}$일 때의 실수 $x$의 값들의 곱을 $\alpha$라 하고,

$z = -\bar{z}$일 때의 실수 $x$의 값들의 곱을 $\beta$라 할 때, $\alpha\beta$의 값을 구하시오.

(단, $i = \sqrt{-1}$이고 $\bar{z}$는 $z$의 켤레복소수이다.)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 조건 | 동치 조건 | 의미 |
|------|-----------|------|
| $z = \bar{z}$ | $\text{Im}(z) = 0$ | $z$는 실수 |
| $z = -\bar{z}$ | $\text{Re}(z) = 0$ | $z$는 순허수 (또는 $0$) |

### 풀이방법

**1단계 — $z = \bar{z}$ 조건** ($z$가 실수, 허수부 $= 0$):

$$5x^2 - 24x - 5 = 0$$
$$(5x + 1)(x - 5) = 0$$
$$x = 5 \quad \text{또는} \quad x = -\frac{1}{5}$$

$z \neq 0$ 확인:
- $x = 5$: $\text{Re}(z) = 25 - 26 = -1 \neq 0$ → 유효
- $x = -\dfrac{1}{5}$: $\text{Re}(z) = \dfrac{1}{25} - 26 \neq 0$ → 유효

$$\alpha = 5 \times \left(-\frac{1}{5}\right) = -1$$

(근과 계수의 관계: $5x^2 - 24x - 5 = 0$의 두 근의 곱 $= \dfrac{-5}{5} = -1$)

**2단계 — $z = -\bar{z}$ 조건** ($z$가 순허수, 실수부 $= 0$):

$$x^2 - 26 = 0$$
$$x = \sqrt{26} \quad \text{또는} \quad x = -\sqrt{26}$$

$z \neq 0$ 확인:
- $x = \pm\sqrt{26}$: $\text{Im}(z) = 5(26) - 24(\pm\sqrt{26}) - 5 = 125 \mp 24\sqrt{26} \neq 0$ → 유효

$$\beta = \sqrt{26} \times (-\sqrt{26}) = -26$$

(근과 계수의 관계: $x^2 - 26 = 0$의 두 근의 곱 $= -26$)

**3단계 — 최종 답**:

$$\alpha\beta = (-1) \times (-26) = 26$$

> **오답 포인트**: ① $z = \bar{z}$를 "$z$가 순허수" 조건으로, $z = -\bar{z}$를 "$z$가 실수" 조건으로 뒤바꾸는 실수(가장 흔함). ② 두 근의 곱을 구할 때 근을 직접 계산하지 않고 근과 계수의 관계를 활용하지 못하는 비효율.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $z = \bar{z}$이면 $z$는 어떤 수인가? $z = -\bar{z}$이면?

**A.** $z = \bar{z} \iff z + \bar{z} - 2z = 0 \iff \text{Im}(z) = 0$이므로 $z$는 **실수**. $z = -\bar{z} \iff z + \bar{z} = 0 \iff \text{Re}(z) = 0$이므로 $z$는 **순허수 또는 0**.

### 카드 2
**Q.** $z = \bar{z}$와 $z = -\bar{z}$를 $z + \bar{z}$, $z - \bar{z}$ 공식으로 유도하는 방법은?

**A.** $z + \bar{z} = 2\,\text{Re}(z)$, $z - \bar{z} = 2i\,\text{Im}(z)$. $z = \bar{z} \iff z - \bar{z} = 0 \iff 2i\,\text{Im}(z) = 0 \iff \text{Im}(z) = 0$. $z = -\bar{z} \iff z + \bar{z} = 0 \iff 2\,\text{Re}(z) = 0 \iff \text{Re}(z) = 0$.

### 카드 3
**Q.** 이차방정식 $ax^2 + bx + c = 0$의 두 근의 곱을 빠르게 구하는 방법은?

**A.** 근과 계수의 관계: 두 근의 곱 $= \dfrac{c}{a}$. 직접 근을 구하지 않아도 된다. 예: $5x^2 - 24x - 5 = 0$의 두 근의 곱 $= \dfrac{-5}{5} = -1$.

### 카드 4
**Q.** 복소수 조건에서 실수부 $= 0$ 또는 허수부 $= 0$을 방정식으로 세울 때 주의할 점은?

**A.** 조건에서 나온 방정식의 근이 "$z \neq 0$" 조건을 위반하는지 확인해야 한다. 실수부 $= 0$인 근 $x_0$에서 허수부도 $0$이면 $z = 0$이 되어 제외.

### 카드 5
**Q.** 복소수 조건 문제에서 자주 하는 실수(오답 패턴)는?

**A.** ① $z = \bar{z}$(실수) vs $z = -\bar{z}$(순허수) 혼동. ② 근과 계수 관계 미활용으로 불필요한 계산. ③ $z \neq 0$ 조건 미확인으로 $z = 0$인 근을 답에 포함. ④ $\alpha$, $\beta$를 "값"이 아닌 "합"으로 오독.
