# 복소수 오답 문제 정리

> 원본 이미지: `SCR-20260328-nrtb.png`

---

## 문제 0491

2023년 06월 고1 학력평가 29번

49 이하의 두 자연수 $m$, $n$에 대하여

$$\left[\left(\frac{1+i}{\sqrt{2}}\right)^m - i^n\right]^2 = 4$$

를 만족시킬 때, $m+n$의 최댓값을 구하시오. (단, $i = \sqrt{-1}$)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| $\dfrac{1+i}{\sqrt{2}}$의 극형식 | $e^{i\pi/4}$ (편각 $45°$, 절댓값 $1$) |
| 거듭제곱 주기 | $\left(\dfrac{1+i}{\sqrt{2}}\right)^8 = 1$ (주기 8) |
| $i$의 거듭제곱 주기 | $i^4 = 1$ (주기 4) |
| 제곱 = 상수 | $A^2 = 4 \Rightarrow A = 2$ 또는 $A = -2$ |

### 풀이방법

1. **$A = \left(\dfrac{1+i}{\sqrt{2}}\right)^m$의 값 분류** ($m \bmod 8$에 따라):

| $m \bmod 8$ | $A$ |
|:-----------:|:---:|
| 0 | $1$ |
| 1 | $\dfrac{1+i}{\sqrt{2}}$ |
| 2 | $i$ |
| 3 | $\dfrac{-1+i}{\sqrt{2}}$ |
| 4 | $-1$ |
| 5 | $\dfrac{-1-i}{\sqrt{2}}$ |
| 6 | $-i$ |
| 7 | $\dfrac{1-i}{\sqrt{2}}$ |

2. **$B = i^n$의 값 분류** ($n \bmod 4$에 따라):

| $n \bmod 4$ | $B$ |
|:-----------:|:---:|
| 0 | $1$ |
| 1 | $i$ |
| 2 | $-1$ |
| 3 | $-i$ |

3. **방정식 분석**: $(A-B)^2 = 4 \Rightarrow A-B = 2$ 또는 $A-B = -2$

   $A-B$가 실수 $\pm 2$가 되려면 $A$와 $B$가 모두 **실수**여야 함.

   - $A$가 실수 조건: $m \equiv 0 \pmod{4}$ → $A = 1$ (m이 8의 배수) 또는 $A = -1$ (m ≡ 4 mod 8)
   - $B = i^n$이 실수 조건: $n \equiv 0 \pmod{2}$ → $B = 1$ ($n \equiv 0 \pmod 4$) 또는 $B = -1$ ($n \equiv 2 \pmod 4$)

4. **경우의 수**:

   | 경우 | $A$ | $B$ | $A-B$ | $m$ 조건 | $n$ 조건 |
   |------|-----|-----|--------|----------|----------|
   | ① | $1$ | $-1$ | $2$ ✓ | $m \equiv 0 \pmod{8}$ | $n \equiv 2 \pmod{4}$ |
   | ② | $-1$ | $1$ | $-2$ ✓ | $m \equiv 4 \pmod{8}$ | $n \equiv 0 \pmod{4}$ |

5. **$m+n$ 최댓값** (49 이하 자연수):

   - 경우 ①: $m$의 최댓값 = 48, $n$의 최댓값 = 46 → $m+n = 94$
   - 경우 ②: $m$의 최댓값 = 44, $n$의 최댓값 = 48 → $m+n = 92$

   $$\therefore\; \text{최댓값} = \boxed{94}$$

> **오답 포인트**: ① $(A-B)^2=4$에서 $A-B=2$만 고려하고 $A-B=-2$를 놓침. ② $A, B$가 허수인 경우도 차가 실수 $\pm2$가 될 수 있다고 착각 (허수부 상쇄 조건 미확인). ③ 49 이하 조건에서 $m=48$이 $m\equiv0\pmod8$을 만족함에도 $m=40$으로 잘못 구함.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $\dfrac{1+i}{\sqrt{2}}$를 극형식으로 나타내면? 그리고 이 수의 거듭제곱 주기는?

**A.** $\dfrac{1+i}{\sqrt{2}} = e^{i\pi/4}$ (절댓값 1, 편각 $\frac{\pi}{4}$). 주기는 8 ($e^{i\pi/4 \cdot 8} = e^{2\pi i} = 1$).

### 카드 2
**Q.** $[A-B]^2 = k^2$에서 $A-B$의 값이 실수가 되기 위한 복소수 $A, B$의 조건은?

**A.** $A$와 $B$의 허수부가 서로 같아서 상쇄되어야 한다. 가장 단순한 경우는 $A, B$ 모두 실수인 경우.

### 카드 3
**Q.** 복소수의 거듭제곱이 주기를 가질 때, $|z|=1$인 복소수의 주기를 구하는 방법은?

**A.** $z = e^{i\theta}$이면 $z^n = e^{in\theta} = 1$이 되는 최소 자연수 $n = \dfrac{2\pi}{\theta}$ (단, 이 값이 자연수인 경우).

### 카드 4
**Q.** 두 조건 $m \leq N$, $m \equiv r \pmod{k}$를 동시에 만족하는 최대 자연수 $m$을 구하는 방법은?

**A.** $N$을 $k$로 나눠 나머지를 확인. 나머지가 $r$보다 크거나 같으면 $m = N - (N\bmod k) + r$, 작으면 $m = N - (N\bmod k) + r - k$.

### 마지막 카드
**Q.** 복소수 거듭제곱의 범위 최적화 문제에서 자주 하는 실수(오답 패턴)는?

**A.** ① $A-B = \pm c$에서 한 부호만 확인. ② 각 케이스에서 $m, n$ 최댓값을 독립적으로 구한 후 합산 (케이스별 조건이 다름을 무시). ③ $(\sqrt{2})^{-1}$을 포함한 허수 항이 차에서 상쇄될 가능성을 검토하지 않음.
