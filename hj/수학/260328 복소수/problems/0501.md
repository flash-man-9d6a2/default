## 문제 0501

0이 아닌 두 실수 $a$, $b$에 대하여 $\dfrac{\sqrt{a}}{\sqrt{b}} = -\sqrt{\dfrac{a}{b}}$일 때, $\sqrt{a} + \sqrt{b}$의 켤레복소수를 구하면?

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 음수의 제곱근 | $x < 0$이면 $\sqrt{x} = i\sqrt{-x}$ (순허수) |
| 제곱근 분리 법칙 | $\sqrt{ab} = \sqrt{a}\sqrt{b}$는 $a, b \geq 0$일 때만 성립 |
| 켤레복소수 | $\overline{p+qi} = p-qi$ |

### 풀이방법

1. **$a > 0, b > 0$인 경우 검토**:

   $\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}} > 0$이고 $-\sqrt{\dfrac{a}{b}} < 0$이므로 등식 불가. ✗

2. **$a < 0, b < 0$인 경우 검토**:

   $\sqrt{a} = i\sqrt{-a}$, $\sqrt{b} = i\sqrt{-b}$

   좌변: $\dfrac{i\sqrt{-a}}{i\sqrt{-b}} = \sqrt{\dfrac{-a}{-b}} = \sqrt{\dfrac{a}{b}} > 0$ (단, $a/b > 0$)

   우변: $-\sqrt{\dfrac{a}{b}}$. $a/b > 0$이므로 우변 $< 0$. 등식 불가. ✗

3. **$a > 0, b < 0$인 경우 검토**:

   $\sqrt{b} = i\sqrt{-b}$ ($= i\sqrt{|b|}$)

   좌변: $\dfrac{\sqrt{a}}{i\sqrt{|b|}} = \dfrac{\sqrt{a} \cdot (-i)}{-i^2 \sqrt{|b|}} = \dfrac{-i\sqrt{a}}{\sqrt{|b|}} \cdot \dfrac{1}{1}$

   더 간단히: $\dfrac{\sqrt{a}}{i\sqrt{|b|}} \cdot \dfrac{-i}{-i} = \dfrac{-i\sqrt{a}}{\sqrt{|b|}} = -i\sqrt{\dfrac{a}{|b|}}$

   우변: $b < 0$이므로 $\dfrac{a}{b} < 0$, 따라서 $\sqrt{\dfrac{a}{b}} = i\sqrt{\dfrac{a}{|b|}}$

   우변 $= -i\sqrt{\dfrac{a}{|b|}}$ → 좌변 $=$ 우변 ✓

4. **$a < 0, b > 0$인 경우 검토**:

   $\sqrt{a} = i\sqrt{|a|}$, $\sqrt{b} = \sqrt{b}$

   좌변: $\dfrac{i\sqrt{|a|}}{\sqrt{b}} = i\sqrt{\dfrac{|a|}{b}}$ (순허수, 양의 허수)

   우변: $a/b < 0$이므로 $-\sqrt{\dfrac{a}{b}} = -i\sqrt{\dfrac{|a|}{b}}$ (순허수, 음의 허수)

   좌변 $\neq$ 우변. ✗

5. **결론**: $a > 0$, $b < 0$

   $\sqrt{a} + \sqrt{b} = \sqrt{a} + i\sqrt{|b|}$ (실수부 $= \sqrt{a} > 0$, 허수부 $= \sqrt{|b|} > 0$)

6. **켤레복소수**:

$$\overline{\sqrt{a}+\sqrt{b}} = \sqrt{a} - i\sqrt{|b|} = \sqrt{a} - i\sqrt{-b} = \sqrt{a} - \sqrt{b}$$

> 단, $b < 0$이므로 $\sqrt{b} = i\sqrt{-b}$임에 주의. 따라서 $\sqrt{a} - \sqrt{b} = \sqrt{a} - i\sqrt{-b}$.

> **오답 포인트**: ① $\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}}$가 항상 성립한다고 가정. ② $a < 0, b > 0$인 경우 검토를 빠뜨려 부호 조건을 잘못 구함. ③ $b < 0$일 때 $\sqrt{b}$를 실수로 계산.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $\sqrt{a} \cdot \sqrt{b} = \sqrt{ab}$, $\dfrac{\sqrt{a}}{\sqrt{b}} = \sqrt{\dfrac{a}{b}}$가 성립하기 위한 조건은?

**A.** $a \geq 0$이고 $b \geq 0$일 때만 성립. 하나라도 음수이면 이 등식이 깨진다.

### 카드 2
**Q.** $a < 0$인 실수에 대해 $\sqrt{a}$를 복소수로 표현하면?

**A.** $\sqrt{a} = i\sqrt{-a} = i\sqrt{|a|}$ (순허수).

### 카드 3
**Q.** 복소수 $z = p + qi$ ($p, q$ 실수)의 켤레복소수를 구하는 방법은?

**A.** $\bar{z} = p - qi$. 허수부의 부호만 바꾼다.

### 카드 4
**Q.** $\dfrac{\sqrt{a}}{\sqrt{b}}$에서 $b < 0$일 때 계산 순서는?

**A.** $\sqrt{b} = i\sqrt{|b|}$로 치환 후 $\dfrac{\sqrt{a}}{i\sqrt{|b|}} = \dfrac{\sqrt{a}}{i\sqrt{|b|}} \cdot \dfrac{-i}{-i} = -i\sqrt{\dfrac{a}{|b|}}$로 유리화.

### 마지막 카드
**Q.** 실수 $a, b$의 부호 조건을 구하는 문제에서 자주 하는 실수(오답 패턴)는?

**A.** ① 모든 경우 (++, --, +-, -+)를 체계적으로 검토하지 않고 한두 경우만 확인. ② 음수를 포함한 제곱근 연산에서 실수 법칙을 그대로 적용. ③ $a > 0, b < 0$과 $a < 0, b > 0$이 대칭이라 착각하여 둘 다 성립한다고 결론냄.
