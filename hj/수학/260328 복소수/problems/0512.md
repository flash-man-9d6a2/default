## 문제 0512

복소수 $z = \dfrac{2-\sqrt{5}\,i}{\sqrt{5}+2i}$에 대하여 $w = \dfrac{1-\bar{z}}{1-z}$라고 할 때, $w$의 값을 구하는 과정을 단계별로 서술하시오.

(단, $i = \sqrt{-1}$이고 $\bar{z}$는 $z$의 켤레복소수이다.)

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 복소수 유리화 | 분모의 켤레복소수를 분자·분모에 곱함 |
| 켤레복소수 | $\overline{a+bi} = a-bi$ |
| 분모의 실수화 | $(a+bi)(a-bi) = a^2+b^2$ |
| $\dfrac{1-i}{1+i}$ 정리 | 분자·분모에 $1-i$를 곱하여 유리화 |

### 풀이방법

**[1단계] $z$ 계산 — 분모 유리화**

분모 $\sqrt{5}+2i$의 켤레인 $\sqrt{5}-2i$를 곱한다.

$$z = \frac{(2-\sqrt{5}\,i)(\sqrt{5}-2i)}{(\sqrt{5}+2i)(\sqrt{5}-2i)}$$

- 분모: $(\sqrt{5})^2 + 2^2 = 5 + 4 = 9$
- 분자 전개:

$$(2-\sqrt{5}\,i)(\sqrt{5}-2i) = 2\sqrt{5} - 4i - 5i + 2\sqrt{5}\,i^2$$
$$= 2\sqrt{5} - 9i - 2\sqrt{5} = -9i$$

$$\therefore\; z = \frac{-9i}{9} = -i$$

**[2단계] $\bar{z}$ 계산**

$$\bar{z} = \overline{-i} = i$$

**[3단계] $w$ 계산**

$$w = \frac{1-\bar{z}}{1-z} = \frac{1-i}{1-(-i)} = \frac{1-i}{1+i}$$

분자·분모에 $1-i$를 곱하여 유리화:

$$w = \frac{(1-i)^2}{(1+i)(1-i)} = \frac{1 - 2i + i^2}{1+1} = \frac{1 - 2i - 1}{2} = \frac{-2i}{2} = -i$$

$$\boxed{w = -i}$$

> **오답 포인트**: 분자 전개 시 $i^2 = -1$을 대입하지 않아 $2\sqrt{5}\,i^2$을 $+2\sqrt{5}$로 처리하지 못하는 실수가 많다. 또한 $w$를 구할 때 분모 $1-z = 1+i$가 됨을 놓치고 $1-i$로 쓰는 오류에 주의한다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** 복소수 분수 $\dfrac{a+bi}{c+di}$를 표준형으로 만드는 방법은?

**A.** 분모의 켤레복소수 $c-di$를 분자·분모에 곱한다. 분모 $= c^2+d^2$ (실수), 분자를 전개할 때 $i^2=-1$을 반드시 대입한다.

---

### 카드 2
**Q.** $z = a+bi$일 때 $\bar{z}$와 $-z$의 차이는?

**A.** $\bar{z} = a - bi$ (허수부 부호만 바꿈), $-z = -a - bi$ (실수부와 허수부 모두 부호 바꿈). 혼동하지 않도록 주의.

---

### 카드 3
**Q.** $\dfrac{1-i}{1+i}$를 간단히 하는 두 가지 방법은?

**A.**
- 방법 1: 분자·분모에 $1-i$를 곱 → $\dfrac{(1-i)^2}{2} = \dfrac{-2i}{2} = -i$
- 방법 2: 분자·분모에 $\dfrac{1}{1+i}$의 켤레 $1-i$를 곱 (동일)

항상 $-i$가 나온다.

---

### 카드 4
**Q.** 복소수 분수식을 단계별로 풀 때 효율적인 순서는?

**A.** ① 주어진 복소수 $z$를 유리화하여 표준형 $a+bi$로 단순화 → ② $\bar{z}$ 도출 → ③ 목표 식에 대입 → ④ 다시 유리화하여 최종 값 도출. $z$를 먼저 단순화하면 이후 계산이 크게 줄어든다.

---

### 카드 5
**Q.** 복소수 유리화 문제에서 자주 하는 실수(오답 패턴)는?

**A.**
- $i^2 = -1$ 대입을 빠뜨려 실수·허수부 분리를 틀림
- 분모 $(a+bi)(a-bi) = a^2+b^2$ 계산에서 부호 오류
- $\bar{z}$와 $-z$, 또는 $1-z$와 $1-\bar{z}$를 혼동하여 대입 실수
