## 문제 0518

실수가 아닌 복소수 $z = (2+i)x^2 + (3i-2)x - 4 + 2i$에 대하여

$z + \bar{z} < 0$을 만족시킬 때, 모든 실수 $x$의 값의 합을 구하시오.

(단, $i = \sqrt{-1}$)

> ※ 추정: 이미지에서 $z + \bar{z} < 0$으로 읽힘. "실수가 아닌" 조건과 부등식을 동시에 만족하는 정수 $x$ 또는 경계값의 합을 구하는 문제로 판단됨.

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| $z + \bar{z}$ | $z + \bar{z} = 2\,\text{Re}(z)$ |
| 부등식 변환 | $z + \bar{z} < 0 \Leftrightarrow \text{Re}(z) < 0$ |
| $z$가 실수가 아닌 조건 | $\text{Im}(z) \neq 0$ |

### 풀이방법

**[1단계] $z$의 실수부·허수부 분리**

$$z = (2+i)x^2 + (3i-2)x + (-4+2i)$$

실수부와 허수부를 모아 정리:

$$\text{Re}(z) = 2x^2 - 2x - 4$$
$$\text{Im}(z) = x^2 + 3x + 2$$

**[2단계] 조건 ① $z + \bar{z} < 0$ 적용**

$$z + \bar{z} = 2\,\text{Re}(z) < 0 \implies \text{Re}(z) < 0$$

$$2x^2 - 2x - 4 < 0$$
$$x^2 - x - 2 < 0$$
$$(x-2)(x+1) < 0$$
$$-1 < x < 2$$

**[3단계] 조건 ② $z$가 실수가 아닌 조건 적용**

$$\text{Im}(z) \neq 0 \implies x^2 + 3x + 2 \neq 0$$
$$(x+1)(x+2) \neq 0$$
$$x \neq -1, \quad x \neq -2$$

$-1 < x < 2$ 범위에서 $x = -2$는 범위 밖이므로 무관. $x = -1$도 범위 밖(경계 불포함). 추가 제약 없음.

**[4단계] 최종 범위 및 합 계산**

$-1 < x < 2$ 안에서 "모든 실수 $x$의 합"이 의미를 가지려면, 문제가 **정수 $x$** 또는 **특정 이산 조건**을 요구하는 것으로 추정.

정수 $x$: $x = 0, 1$ → 합 $= 0 + 1 = \boxed{1}$

> ※ 추정: 원본 문제에서 "정수 $x$" 또는 범위 내 특정 조건이 추가로 있을 수 있음. 원본 확인 권장.

> **오답 포인트**: 복소수에 직접 부등호를 적용하려는 실수. $z + \bar{z} < 0$을 $z < 0$으로 간주하면 안 된다. 반드시 $2\,\text{Re}(z) < 0$으로 변환하여 실수 부등식으로 다루어야 한다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** $z + \bar{z} < 0$, $z - \bar{z} > 0$ 등 켤레복소수 부등식을 실수 조건으로 변환하는 방법은?

**A.**
- $z + \bar{z} = 2\,\text{Re}(z)$ → 조건: $\text{Re}(z) < 0$
- $z - \bar{z} = 2i\,\text{Im}(z)$ → 이건 허수이므로 직접 부등식 불가; 실수부·허수부 분리 후 판단

복소수 부등식은 반드시 실수부/허수부로 분리하여 실수 부등식으로 변환한다.

---

### 카드 2
**Q.** 복소수 $z = f(x) + g(x)i$ (실수 $x$에 대한 함수)에서 실수부·허수부를 올바르게 분리하는 방법은?

**A.** 각 항의 계수를 실수 성분과 허수 성분으로 분리한다. 예: $(a+bi)x^2$의 실수부 $= ax^2$, 허수부 $= bx^2$. 전체를 모아 $\text{Re}(z)$, $\text{Im}(z)$를 $x$의 다항식으로 표현한다.

---

### 카드 3
**Q.** "$z$는 실수가 아니다"와 부등식 조건을 동시에 만족하는 해를 구하는 전략은?

**A.** ① 부등식 조건 → 해 범위 $A$ 구하기, ② $\text{Im}(z) \neq 0$ → 제외 집합 $B$ 구하기, ③ 최종 해 $= A \setminus B$. 두 조건을 각각 따로 풀고 교집합·차집합으로 처리한다.

---

### 카드 4
**Q.** 이차부등식 $(x-p)(x-q) < 0$ ($p < q$)의 해는?

**A.** $p < x < q$. 등호가 없으면 경계값 $p, q$ 제외. 등호가 있으면 경계값 포함.

---

### 카드 5
**Q.** 복소수 부등식 문제에서 자주 하는 실수(오답 패턴)는?

**A.**
- $z + \bar{z} < 0$을 $\text{Re}(z) < 0$이 아닌 $z < 0$으로 처리
- 실수부·허수부 분리 시 허수 계수의 실수 기여분을 잘못 계산 (예: $i^2 = -1$ 누락)
- "$z$가 실수가 아님" 조건을 부등식 풀이에 반영하지 않아 불필요한 해를 포함하거나 범위가 잘못됨
