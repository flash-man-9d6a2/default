## 문제 0522

2024년 03월 고2 학력평가 15번

다음 조건을 만족시키는 복소수 $z$가 존재하도록 하는 모든 실수 $k$의 값의 곱은?
(단, $\bar{z}$는 $z$의 켤레복소수이다.)

> ※ 이미지에서 조건 부분이 잘려 정확한 조건 확인 불가. 손필기 메모에 "$= -(1)$" 형태가 보이므로, 켤레복소수 조건이 포함된 방정식에서 실수 $k$를 구하는 유형으로 추정.

---

## 핵심 공식 & 풀이방법

### 핵심 공식

| 개념 | 공식 |
|------|------|
| 복소수 분리 | $z = a+bi$ ($a, b \in \mathbb{R}$)로 놓고 실수부·허수부 분리 |
| 켤레복소수 | $\bar{z} = a-bi$ |
| 켤레복소수의 합·곱 | $z+\bar{z} = 2a$, $z\bar{z} = a^2+b^2$ |
| 복소수 존재 조건 | 실수부 = 0, 허수부 = 0인 연립방정식이 실수해 $(a,b)$를 가질 것 |
| 이차방정식 판별식 | 실수해 존재: $D = b^2 - 4ac \geq 0$ |
| 근과 계수의 관계 | 두 근의 곱 $= \dfrac{c}{a}$ |

### 풀이방법

1. $z = a + bi$ ($a, b \in \mathbb{R}$)로 놓는다.
2. 주어진 조건을 $a$, $b$에 대한 식으로 변환하고, 실수부와 허수부를 분리하여 연립방정식을 세운다.
3. 연립방정식이 실수 해 $(a, b)$를 가질 조건 → $k$에 관한 부등식 또는 방정식 도출.
4. 경계 조건(판별식 $D = 0$ 또는 특정 값)에서의 $k$ 값들을 구한다.
5. "모든 $k$ 값의 곱" → 근과 계수의 관계(비에타 공식) 적용.

> **오답 포인트**: $z$를 $a+bi$로 분해하지 않고 추상적으로 다루면 조건을 실수 방정식으로 변환할 수 없다. 또한 "존재하도록 하는"의 의미를 $D \geq 0$으로 정확히 연결하지 못하는 실수가 많다.

## 플래시카드

> 카드는 **이 문제에 국한된 질문이 아니라**, 유사한 문제에도 재사용할 수 있는 범용적인 질문으로 구성합니다.

### 카드 1
**Q.** 복소수 $z$가 존재한다는 조건을 수학적으로 어떻게 처리하는가?
**A.** $z = a+bi$ ($a, b \in \mathbb{R}$)로 놓은 뒤, 주어진 식의 실수부 $= 0$, 허수부 $= 0$인 연립방정식이 실수 해 $(a, b)$를 가져야 한다는 조건으로 전환한다.

### 카드 2
**Q.** 켤레복소수 $\bar{z}$가 포함된 식을 풀 때 핵심 전략은?
**A.** $z = a+bi$로 놓으면 $\bar{z} = a-bi$이므로, $z\bar{z} = a^2+b^2$, $z+\bar{z} = 2a$처럼 실수 식으로 변환된다. 이를 이용해 복소수 방정식을 실수 연립방정식으로 바꾼다.

### 카드 3
**Q.** 매개변수 $k$를 포함한 방정식에서 "실수 해가 존재하는 $k$의 범위 또는 값"을 구하는 방법은?
**A.** 해가 존재할 경계 조건은 판별식 $D = 0$이다. $D \geq 0$인 $k$ 범위, 또는 $D = 0$이 되는 $k$ 값을 구한다. "모든 $k$의 곱"은 비에타 공식으로 처리한다.

### 카드 4
**Q.** 이차방정식 $ax^2+bx+c=0$의 두 근의 곱을 비에타 공식으로 나타내면?
**A.** 두 근의 곱 $= \dfrac{c}{a}$. "모든 $k$의 값의 곱"을 구할 때, $k$에 관한 방정식을 세우고 상수항을 최고차항 계수로 나누면 된다.

### 카드 5
**Q.** 복소수 조건 문제에서 자주 하는 실수(오답 패턴)는?
**A.** ① $z$를 $a+bi$로 분해하지 않고 복소수 전체로 계산하려는 것. ② 실수부·허수부를 각각 0으로 놓는 것을 빠뜨리는 것. ③ "복소수 $z$가 존재"를 이차방정식의 판별식 $D \geq 0$과 연결하지 못하는 것.
